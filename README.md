<h1 align="center">
    <b>Kurulum</b>
</h1>


Bilgisayarınızda **mpv** programının kurulu olduğundan emin olun. Eğer kurulu değilse, aşağıdaki adımları izleyerek kurabilirsiniz:


**Arch için:**
```bash
sudo pacman -S mpv
```
Ubuntu/Debian
```bash
sudo apt-get install mpv
```
